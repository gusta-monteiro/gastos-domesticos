# Gastos Domésticos
A personal finance dashboard to track monthly income and expenses across custom categories, with period comparisons and AI-generated financial analysis.

## Features
- Monthly budget tracker with customizable category allocations
- Expense logging per category with item-level detail
- Period overview with income vs. spending charts
- Printable reports with AI-powered financial insights (Claude API)
- Data persisted locally via localStorage — no backend required

## Stack
HTML · CSS · JavaScript · Chart.js · Anthropic API

## Usage
Open `index.html` directly in any browser. No build step required.
To enable AI analysis, add your Anthropic API key in the settings within the app.
